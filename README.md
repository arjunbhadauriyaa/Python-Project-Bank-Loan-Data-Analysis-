# Python-Project-Bank-Loan-Data-Analysis-

# Overview
This project focuses on analyzing bank loan data using Python. The main goal of the project is to understand the bank’s lending performance, customer repayment behavior, monthly loan trends, and overall loan portfolio performance.

The analysis was performed using a real-world loan dataset in Jupyter Notebook with the help of Python libraries such as Pandas and Matplotlib.
# This project demonstrates practical skills in:

Data Cleaning
Data Analysis
KPI Calculation
Business Analysis
Data Visualization
Loan Performance Analysis

# The main objectives of this project are:

Analyze total loan applications
Analyze month-to-date (MTD) loan applications
Calculate total funded amount by the bank
Calculate total amount received from customers
Compare funded amount and received amount
Analyze good loans and bad loans
Analyze loan status distribution
Calculate average interest rate
Calculate average debt-to-income ratio (DTI)
Analyze monthly loan trends
Create charts and visualizations for business understanding
